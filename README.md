# obsidian-to-hugo
This is a simple script to convery my YAML frontmatter in obsidian notes to TOML for posting on HUGO 

## Installation & Usage:

```sh
# Activate the venv
source .venv/bin/activate
```

- Set Obsidian and Hugo folders.
- Run the script:

```sh
python convert.py
```

- Converted files will be in hugo_notes/
